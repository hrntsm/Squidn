# 変形図の床・剛床追従とスケール改善 申し送り

作成日: 2026-07-23
対象コード: `crates/squid-n-app/src/viewer/mod.rs`、`crates/squid-n-app/src/app/mod.rs`
利用者向け計算根拠: `docs/result_view.md`（変形図の表示倍率・床/剛床の追従を更新）

梁の変形図を Hermite 3 次曲線で描く実装（`梁内部の変位応力内挿_申し送り.md`）を
入れた結果、床・二次部材・剛床代表点が梁曲線に追従しない・変形スケールが梁で
過大になる、といった見え方の問題が出ていた。次の 5 点を修正した。

## 1. 床・二次部材の節点を梁の Hermite 変形曲線へ載せる

### 症状（変更前）

`interpolate_unreferenced_disp`（未参照節点の変位補間）は、大梁に直付きする節点を
「線分上の射影位置 $t$ での端点変位の**線形補間**」で追従させていた。一方で梁自体は
端部回転を含む Hermite 3 次曲線で描かれるため、スラブ境界・小梁支持点・二次部材の
節点が**梁の描画曲線から浮いて**（弦の上に取り残されて）見えた。

### 対応（変更後）

Hermite 変位評価を `beam_hermite_disp_at`（無倍率のグローバル並進変位を材軸
パラメータ ξ で返す）に切り出し、曲線描画（`beam_deformed_polyline`）と補間の
両方で共有した。補間の直付き判定に要素種別を持たせ、

- **梁（`Beam`）に直付き**する節点: 並進 3 成分を梁の Hermite 変位（射影位置 $t$）
  で追従（回転は端点の線形補間）。同じ関数を使うため節点は梁の描画曲線に厳密一致
  する（テスト `梁上の未参照節点は梁の描画曲線に厳密一致する`）。
- **梁以外の線材**（ブレース等）に直付き: 従来どおり端点変位の線形補間。

二次部材の接続を辿る追従（段階 2）・孤立節点のフォールバック（射影）は従来どおり。
あくまで描画専用で `StaticOnce::disp` は変更しない。

## 2. 剛床代表節点の鉛直変位を表示用に補完（床面への追従）

剛床代表節点（重心マスター）の面内自由度（$U_x$・$U_y$・$R_z$）は解析結果を持ち、
`coords3` に加味されるため水平変形には従来から追従していた。一方、面外自由度
（$U_z$・$R_x$・$R_y$）は零剛性による特異行列を避けるための数値ダミー拘束で 0 に
固定されるため、重力たわみや地震の転倒による床の上下動へ追従せず、代表点だけが
原標高に浮いて見えていた。

`fill_diaphragm_master_disp_for_display` を追加し、代表点の $U_z$ を**スレーブ節点の
$U_z$ 平均**で表示用に補う（描画専用、解析結果は不変）。面内成分は解析値のまま。
テスト `剛床マスターの鉛直変位はスレーブ平均で補完される`。

## 3. 剛床代表点のスレーブ点線接続と表示トグル

- `App::show_diaphragm_master`（既定 OFF）を追加。ON で剛床代表点を可視化する。
  - 代表点（重心の仮想節点）を強調リングで示す。
  - 代表点から関連スレーブ節点へ**点線**（変形後座標）を引き、所属関係を可視化。
  - 面内拘束マーク（$U_x$・$U_y$・$R_z$）と左下凡例もこのトグルに連動させた
    （従来は常時表示）。
- 点線はスレーブが多い階でスターバースト状になり他部材が見づらいため、既定は
  非表示。剛床がある場合のみツールバーに「剛床代表点」トグルを出す。

## 4. 変形倍率の手動スライダー（自動倍率への係数）

倍率スライダーは以前「変位自動正規化」導入時に一度撤去したが、拡大・縮小の要望に
応えるため**自動倍率への係数**として復活させた。

- `App::deform_scale_factor`（既定 1.0）。実効倍率 = 自動倍率 × 係数。
- 変形を表示するモード（変形・モード・N/Q/M の変形重ね）で「変形倍率」スライダー
  （0.1〜10、対数、「リセット」で 1.0）を表示。
- 左上注記は $f=1$ で「変形倍率 ×N（自動）」、それ以外は「×N（自動×f）」。

## 5. 梁スパン基準の変形スケール上限

`beam_deflection_scale_limit` を追加。各梁の**内部たわみ**（Hermite 曲線の、変形後
両端を結ぶ弦からの最大逸脱。無倍率で 9 分割サンプル）が**スパンの 10%**
（`BEAM_DEFLECTION_DISPLAY_FRAC`）を超えない倍率上限を求め、全梁の最小を採る。
自動倍率は従来のバウンディングボックス基準（最大変位＝対角長 10%）と、この梁基準の
**小さい方**にした。端部回転の大きい梁で中央のふくらみが過大になるのを抑える。
テスト `梁の内部たわみで変形スケール上限が算定される`（L=6000・θz=±0.01 で上限 40.5）。

## テスト

`viewer/mod.rs`（`--features gui`）に追加・更新:

- `大梁スパン中間の未参照節点は梁のHermite変位で追従する`（旧「線形補間になる」を
  Hermite 期待値へ更新。材軸直交成分が $N_3$ 倍になる）
- `梁上の未参照節点は梁の描画曲線に厳密一致する`（補間と `beam_deformed_polyline`
  の一致）
- `剛床マスターの鉛直変位はスレーブ平均で補完される`
- `梁の内部たわみで変形スケール上限が算定される` / `変位ゼロなら梁スケール上限は無し`

既存の射影・二次部材追従・剛床マスター非上書き・端部一致・端部回転たわみの各
テストは維持（軸方向のみの変位ケースは Hermite でも線形と一致するため不変）。

## 6. 内部たわみ表示（内挿）のトグルと N/Q/M 図への適用

「全体の変形を見たい」「梁の内部たわみを見たい」の 2 用途を切り替えられるよう、
梁の内挿表示をトグル化した。

- `App::show_beam_interpolation`（既定 **ON**）と「内部たわみ」トグル（変形表示
  オプション行、変形倍率スライダーと同じ行）を追加。
- **ON**: 梁を Hermite 3 次曲線で描き、床・二次部材も曲線へ追従（`interpolate_
  unreferenced_disp` に `use_beam_hermite` を渡し梁アンカーを Hermite 変位で追従）。
  変形スケールは梁スパン基準も併用（`beam_deflection_scale_limit`）。
- **OFF**: 梁を直線（弦）で描き、床・二次部材の追従も線形補間。スケールは
  バウンディングボックス基準のみ（直線ではふくらみが生じないため）。

あわせて、従来は変形図・モード形でしか曲線描画していなかった梁の Hermite 曲線を、
**N/Q/M 図の変形重ね（`overlay_deform`）**でも描くようにした。曲線描画の判定を
`matches!(mode, Deformed|Mode)` から「変位が表示されている（`disp` が `Some`）
かつトグル ON」に変更したことで、変位が乗る全モード（変形・モード・応力重ね）で
一貫して内挿の有無を切り替えられる。応力図（N/Q/M の張り出し）自体は従来どおり
変形後の材軸（節点間の弦）に沿って描くため、梁曲線とは独立（曲線に沿わせては
いない）。

テスト追加: `大梁スパン中間の未参照節点は内部たわみオフで線形補間になる`
（OFF で線形補間へ戻ることを確認）。

## 7. 応力図（N/Q/M）の張り出しを梁の変形曲線に沿わせる

内部たわみ表示が有効なとき、N/Q/M 図の張り出し（`draw_force_diagram`）の基準線
（張り出し量 0 の線）を、従来の「変形後の節点間直線（弦）」から**梁の変形後
Hermite 曲線**に変更した。梁の線描画と同じ `beam_deformed_point_at`（新規、
`beam_deformed_polyline` もこれを使うよう共通化）で基準点を評価するため、応力図の
基準線が梁の描画曲線に厳密一致する。

- `draw_force_diagram` に `disp: Option<&[[f64;6]]>` と `deform_scale` を渡し、
  内部たわみ ON かつ梁かつ変形重ね（`disp` が Some）のときのみ曲線基準にする。
  それ以外（梁以外・内部たわみ OFF・変形重ね無し）は従来どおり直線基準。
- 張り出し方向（要素ローカル y 軸 `ey`）と応力値・正規化は不変。基準線のみ曲線化。
- テスト追加: `梁変形後曲線の端点は節点変位に一致し中央は弦から外れる`
  （`beam_deformed_point_at` の端点一致とたわみを検証）。

## 8. 結果可視化のリファクタリング（レビュー反映）

梁の変形・応力表示が一通り揃った段階で、結果可視化（`viewer/`）全体をレビューし
構造を整理した。挙動は不変（純粋なリファクタリング＋防御的修正）。

- **`BeamDeflection` 構造体**（`viewer/mod.rs`）: `beam_hermite_disp_at` /
  `beam_deformed_point_at` / `beam_deformed_polyline` の 3 自由関数を、要素ローカル系
  と局所端部 DOF を一度だけ構築して保持する構造体（`new` / `disp_at` / `point_at` /
  `polyline`）へ集約。応力図の張り出しサンプルや `beam_deflection_scale_limit` の
  ループで `LocalFrame::from_nodes` と `rotate_to_local`（毎回 12×12 のヒープ確保）を
  梁 1 本あたり 1 回に削減。上記 §1・§5・§7 の記述中の関数名は本構造体のメソッドに
  置き換わっている。
- **`Projector` 構造体**（`viewer/mod.rs`）: 描画関数群へ引き回していた投影文脈
  `(center3, cam, scale, screen_center)` を束ね、`project` / `project_offset` /
  `cam_space` / `cam_to_screen` をメソッド化。`viewer_panel` で 1 度構築し共有する。
  投影由来の `#[allow(clippy::too_many_arguments)]` を撤去し、`project` の戻り値を
  `Pos2` に統一（`pts: Vec<Pos2>` 化で `egui::pos2(pts[i][0], pts[i][1])` の再ラップを
  一掃）。`solid.rs` の `cam_space` / `to_screen`（`project` 前半・後半の複製）も
  `Projector` へ集約。自由関数 `project` は `mn_view` 用の下位ラッパとして残置。
- **ピッキング集約**: 節点探索（作成モード 3 箇所）・部材探索（クリック選択・
  ホバー 2 箇所）のコピペを `pick_nearest_node` / `pick_nearest_member` へ統合。
- **変位加工・倍率算定の関数抽出**: `display_disp`（未参照節点補間＋剛床代表点補完）と
  `deform_display_scale`（実効表示倍率）を `viewer_panel` から抽出し単体テスト化。
- **図面描画の共通化**: N/Q/M 図と CMQ 図で重複していた閾値定数
  （`MIN_DIAGRAM_PX` / `MIN_SEGMENT_PX`）を `diagram` の `pub(super)` 定数へ一本化。
- **NaN 安全化**: 浮動小数の `partial_cmp().unwrap()` を `unwrap_or(Equal)` に統一
  （検定比・応力サンプル・CMQ サンプル位置。退化した設計検定の 0/0 での panic を回避）。

### リファクタリングで見送った項目（将来対応）

レビューで挙がったが、挙動変更や広範囲な影響を伴うため本 PR では見送った:

- **CMQ M 図の塗り方式**: N/Q/M 図は非凸・符号反転に対応する `diagram_fill_polygons`
  へ移行済みだが、CMQ M は旧来の単一 `convex_polygon`（`paint_diagram_polygon`）の
  まま。複数荷重で非凸になる場合に塗り領域が僅かにずれる余地があり、`diagram_fill_
  polygons` への載せ替えで力図とパリティを取れる（輪郭は別描画のため線は現状も正）。
- **`NodeId::index()` の位置解決**: 大半のパスは `nodes[i].id == NodeId(i)` の不変条件に
  依存して直接添字する一方、接合部検定の着色だけは id→位置の走査を行っており扱いが
  不統一。`Model::node_pos(NodeId)` 等で一本化するか不変条件を明文化する余地がある。
- **検定比凡例の閾値ハードコード**: 凡例スウォッチが 0.8/1.0 と配色を `status_color`
  とは別に直書きしている。`theme` 側に帯テーブルを置いて両者で共有すると同期漏れを
  防げる。
- 低レベル幾何ヘルパ（`cross3` / `norm3` / `normalize3` / `lerp3`）の全面統一、
  `max_ratio_for_kind` の畳み込み共通化、ViewCube の退化四角形ガードなど軽微項目。

## 残課題・改善余地

- 梁たわみの Hermite 内挿は端部 6 自由度のみで、部材中間荷重（UDL）による 4 次の
  ふくらみは表現しない（`梁内部の変位応力内挿_申し送り.md` の残課題と同じ）。床節点も
  同じ端部 6 自由度の曲線に載るため、この近似の範囲で追従する。
- 剛床代表点の鉛直補完はスレーブ $U_z$ の単純平均。床が大きく傾く場合に重心位置
  での平面フィット値と厳密には一致しないが、代表点表示としては十分（$R_x$・$R_y$ は
  描画に使わないため補完しない）。
- 梁スパン基準のスケール上限は全梁の最小を採るため、極端に短い・端部回転の大きい
  梁 1 本が全体倍率を強く律速し得る。その場合は手動スライダーで拡大して調整する
  想定。スパン比の閾値 0.1 は `BEAM_DEFLECTION_DISPLAY_FRAC` で調整可能。
